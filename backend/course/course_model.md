# 교과목 (Course)

- **개설연도** (year)
- **학기** (semester)
  - `1`, `S`, `2`, `W`
- **교과목 코드** (code) 
- **강좌번호** (number) : 3-digit
- 교과목명 (title)
- 학점 (credit) : int
- 교과구분 (category)
  - `교양`, `교직`, `논문`, `대학원`, `일선`, `전선`, `전필`
- 강의 언어 (language)
- 영역 (area)
- 하위영역 (subarea)
- 개설대학 (collage)
- 개설학과 (dept) - nullable
